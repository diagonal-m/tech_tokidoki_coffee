## コンピュータの 5 大装置

インプットされた情報を処理して適切なアウトプットをする一連の動きができる

- **入力**
  - 外部からデータを入力
- **出力**
  - 処理したデータを出力
- **演算** ... CPU
  - 入力されたデータの処理  
    ※ 記憶装置に入ったデータを持ってくる
- **記憶**
  - データを記憶する役割(メモリ)
- **制御** ... CPU
  - 全体の動きのコントロール

## IC(集積回路) の仕組み

> IC(Integrated Circuit)・・・電子部品で作られた論理回路や演算回路

数ミリの小さな基盤の上に、膨大な数のトランジスタ、ダイオード、抵抗がある  
→ **論理回路や半加算器などがぎっしりつまっている**  
→ **IC がたくさん集まることによって CPU：演算・制御装置、メモリ：記憶装置 を実現**

## CPU の４つの機能

> CPU(Central Processing Unit)は主に、演算装置・レジスタ・制御装置・クロックの 4 つの部品から構成されている。

### 演算装置

**0 と 1 の数字を計算して情報を処理する**  
→ 半加算器、全加算器など演算回路がぎっしりと詰まっている

### レジスタ

**「CPU の中で」命令やデータを記憶する役割**  
※ レジスタにあるデータしか CPU は処理できない  
→ メモリからレジスタにデータを持ってこないといけない

> なぜメモリからレジスタに、データを持ってくる？

→ CPU が素早く計算を行うため

### 制御装置

**データのやり取り、演算装置への指示**  
→ 演算ややレジスタにできない「データのやり取り」をしている

### クロック

**一定の間隔で信号を出して、作業のタイミングを合わせる**  
→ 作業の開始と終わりの合図を出して、全体の動きのタイミングを合わせるようにしている。

## RAM と ROM

メモリ：データを保管する場所

- メモリ
  - RAM
    - SRAM
    - DRAM
  - ROM
    - マスタ ROM
    - PROM

### ROM(Read Only Memory)

読み込み専用のメモリ・・・データを保存するだけでそれ以外の作業はできない

### RAM(Random Acccess Memory)

任意のメモリにアクセス・・・データを読み書きするための作業机

ストレージ: 265GB  
RAM: 32GB
